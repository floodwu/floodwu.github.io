```
(E1 + E2 + E3 + E4 + E5)/5 = MA5
```

例：
```
5.66 + 5.88 + 6.47 + 7.12 + 7.3 = 32.43
32.43/5 = 6.486

S4 = 25.13
E4 = 7.12
GR5 = 0.0253
E5 = (1 + GR5)*E4
   = (1 + 0.0253)*7.12
   = 7.3
```

因为：
```
E5 = (1 + GR5)*E4
(E5 + S4)/5 = MA5
```
若要：
```
MA5 = E5
```

则有：
```
((1 + GR5)*E4 + S4)/5 = (1 + GR5)*E4
(1 + GR5)*E4 + S4 = (1 + GR5)*E4*5
E4 + E4*GR5 + S4 = E4*5 + GR5*E4*5
E4 + S4 - E4*5 = GR5*E4*5 - E4*GR5
S4 - 4*E4 = 4*E4*GR5
(S4 - 4*E4)/(4*E4) = GR5
S4/(4*E4) - 1 = GR5
```

即：
```
GR5 = S4 / (4 * E4) - 1
```

例：
```
S4 = 25.13
E4 = 7.12
GR5 = 25.13/(4*7.12) - 1
   = 25.13/28.48 - 1
   = -0.1176
```


买入参考价：
```
B = (S4 / (4 * E4)+ 0.01) * E4；
```


**收盘后看到的MA5是用收盘价计算出来的，盘中的MA5最终没有痕迹。**
