# ~运算符在逻辑判断中的作用

```javascript
console.log(!0);	// true	
console.log(!1);	// false
console.log(!2);	// false
console.log(!-1);	// false

console.log(~0);	// -1
console.log(~1);	// -2
console.log(~2);	// -3
console.log(~-1);	// 0

console.log(!~0);	// false
console.log(!~1);	// false
console.log(!~2);	// false
console.log(!~-1);	// true
```
