# 可以用单个实参来调用的构造函数定义了从形参类型到该类类型的一个隐式转换

可以用单个实参来调用的构造函数定义了从形参类型到该类类型的一个隐式转换：

```C++
class Sales_item {
  public:
    // default argument for book is the empty string
    Sales_item(const std::string &book = ""):
      isbn(book), units_sold(0), revenue(0.0) { }
    Sales_item(std::istream &is);
    // as before
};
```
