# 有哪些通过PHP在Web服务器上执行命令的方式？

有4种主要的在Web服务器上执行命令的方法：
1）`exec()`函数
没有直接的输出，将返回命令执行结果的最后一行
2）`passthru()`函数
直接将输出显示到浏览器，如果输出是二进制，比如图片，这将是非常有用的。
3）`system()`函数
将命令的输出回显到浏览器，它将每一行的输出向后对齐。
