# 异常处理

在PHP中，异常必须手动抛出。

```php
try{
		throw new Exception("An exception",10); // 构造内置Exception对象，参数为异常信息和编号
	}
	catch(Exception $e){
		echo $e->getCode()."<br/>"; 
		echo $e->getMessage()."<br/>";
		echo $e->getFile()."<br/>";
		echo $e->getLine()."<br/>";
		echo $e;
	}
```
输出：
10
An exception
D:\wamp\www\test.php
4
exception 'Exception' with message 'An exception' in D:\wamp\www\test.php:4 Stack trace: #0 {main}
